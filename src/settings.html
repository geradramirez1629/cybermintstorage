Cybermint.Storage

This is an application for Cybermint.Storage, a service for decentralized data storage.

Cybermint.Storage is a service to make building on top of Filecoin as simple as possible - giving the developers the power of open.

Behind the scenes, Cybermint.Storage is backed by Filecoin and makes content available via IPFS leveraging the unique properties of each network.

See the documentation to learn more!
